# agri-data-analysis
프로젝트 제목: 🥬 농산물 조달 리스크 분석 대시보드

프롬프트 링크: (새 창에서 대화한 뒤 생성할 공유 링크)

데이터 설명: "공공데이터포털 가격 데이터와 기상청 강수량 데이터를 활용함"

전처리 과정: "엑셀 날짜 형식 오류(1970년)를 해결하기 위해 pd.to_datetime 로직 보완"

차트 설명: 각 차트가 무엇을 의미하는지 간단히 기술.

품목 선정 이유: 기상 변화에 민감한 엽채류(배추)와 수급 조절이 용이한 저장 채소(양파)의 가격 형성 메커니즘 차이를 비교 분석하기 위해 선정함.
